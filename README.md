## ğŸ“ Customization

To personalize this portfolio for your own use:

1. **Update Personal Information**: 
   - Edit `components/Hero.tsx` for your name and title
   - Update `components/About.tsx` with your bio
   - Modify `components/Contact.tsx` with your contact details

2. **Add Your Projects**:
   - Edit the `projects` array in `components/Projects.tsx`
   - Add real GitHub links and demo URLs

3. **Update Skills**:
   - Modify `components/Skills.tsx` to reflect your skill levels

4. **Add Experience**:
   - Update `components/Experience.tsx` with your actual experience and education

5. **Update Metadata**:
   - Edit `app/layout.tsx` to update SEO information

See `CUSTOMIZE.md` for a detailed customization checklist.
